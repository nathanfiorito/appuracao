## Módulos principais

### `backend/lambdas/`

Cada arquivo corresponde a um passo do Step Functions Express. Todos seguem a mesma interface:

```python
def handler(event: dict, context) -> dict:
    ...
```

O `event` recebe o estado atual da execução; o retorno é mergeado ao estado para o próximo passo.

### `infrastructure/template.yaml`

Template AWS SAM com todos os recursos: tabelas DynamoDB, bucket S3, distribuição CloudFront, funções Lambda, máquina de estados Step Functions, API Gateway REST, roles IAM.

### `infrastructure/statemachine/bulletin_processing.asl.json`

Definição da máquina de estados em Amazon States Language (ASL). Cada estado invoca uma Lambda via `Task` com retry automático em erros transitórios.
